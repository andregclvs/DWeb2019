Abstract:

O sexto trabalho prático consistiu na criação de aplicação para arquivos sonoros em que fosse possível seguir modelo CRUD .

Para a listagem de todos os registos basta aceder a página inicial e aparecerão todos os registos guardados até ao momento.

Para atualização de um registo já existente o utilizador deverá carregar em "Ver" para que as suas informações apareçam no formulário e depois disso poderá alterar todos os campos à exceção do campo Título visto que este funciona como identificador único da música.

Deliverables:

  • XML do Project Record
  • Tratamento de todos os pedidos que chegam ao servidor
  • Layout da aplicação
  • Ficheiro da Base de Dados
  • Função para ver um registo individual
  • Função para apagar um registo
  • Função para atualizar um registo
